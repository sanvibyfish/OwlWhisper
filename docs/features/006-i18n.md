# FEAT-006: 国际化

> 状态：✅ Done | 创建日期：2026-03-17 | 完成日期：2026-03-17

---

## 1. 功能概述

支持英文（默认）和中文两种语言，可在设置窗口切换。

## 2. 用户流程

1. 设置窗口 → Language 下拉框 → 选择 English / 中文
2. 弹窗确认重启 → 重启后生效

## 3. 技术实现

- **Localizable.strings**：`en.lproj/` + `zh-Hans.lproj/`
- **辅助函数**：`private func L(_ key: String) -> String { NSLocalizedString(key, comment: "") }`
- **语言切换**：`UserDefaults.standard.set([lang], forKey: "AppleLanguages")`
- **覆盖范围**：设置窗口全部文字、菜单项、下载进度、更新弹窗

## 4. 注意事项

- 切换语言需要重启 app 才能生效
- 默认跟随系统语言，手动切换会覆盖系统设置
- 两个 strings 文件的 key 必须完全一致
